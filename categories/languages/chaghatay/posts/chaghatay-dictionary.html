Chaghatay Dictionary

#Chaghatay #Dictionary #Tool
2026/6/6

Overview

A small tool website for looking up Chaghatay words.

Notes

The dictionary is based on A Dictionary of Early Middle Turkic, by Hendrik Boeschoten, with the editorial assistance of John O'Kane. I digitized it so it can be searched and used as a practical reference.

I first wanted to show both Latin transliteration and Arabic-script Chaghatay forms. Because the source dictionary uses broad transcription, and because I am still learning Chaghatay orthographic rules, the Arabic-script notes are only temporary references.

For the same reason, the built-in Latin-to-Chaghatay converter is not an exact scholarly converter. It only performs a simple one-to-one letter conversion for quick reference.

Features

  • Full-text search and fuzzy lookup
  • Latin transliteration to Chaghatay script helper for reference only
