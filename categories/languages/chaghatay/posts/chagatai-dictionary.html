Chagatai Dictionary

A small website for looking up Chagatai words.

The data is based on Hendrik Boeschoten's A Dictionary of Early Middle Turkic. This note records the project as a Chaghatay article in the language section.

Overview

This project aims to build an online dictionary for Chagatai. It focuses on clear word lookup, readable forms, and references from early Turkic sources. The dictionary is still under development.

Structure

Search

The tool supports looking up words by headword, transliteration, and visible entry text.

Entry View

Each result keeps the headword, Arabic script form, page reference, and explanatory text close together.

Reading Notes

Short notes stay beside the entries, so the dictionary can work as both a lookup tool and a reading aid.

Current Points

  • Search Chagatai words and their meanings
  • View word forms and grammatical information
  • Check source references from early Turkic texts
  • Keep the project continuously updated and improved
