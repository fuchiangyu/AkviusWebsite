Overview

A small tool website for looking up Chaghatay words.

Notes

The dictionary is based on A Dictionary of Early Middle Turkic, by Hendrik Boeschoten, with the editorial assistance of John O'Kane. I digitized it to make searching and reference easier.

I originally wanted to present Latin transliteration and Arabic-script Chaghatay together. Because the source dictionary uses broad transcription, and because I am still learning Chaghatay spelling rules, the Arabic-script Chaghatay notes are only temporary references.

For the same reason, the built-in Latin-to-Chaghatay converter is not exact. It only performs a simple one-to-one letter conversion for quick reference.

Features

  • Full-text search and fuzzy lookup
  • Latin transliteration to Chaghatay script helper for reference only
